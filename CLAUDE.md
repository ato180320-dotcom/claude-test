# CLAUDE.md

이 파일은 이 저장소에서 작업하는 Claude Code(claude.ai/code)에게 안내를 제공합니다.

## 프로젝트 개요

순수 정적 HTML/CSS/JS 프로젝트입니다. 빌드 단계, 패키지 매니저, 번들러가 없습니다. `.html` 파일을 브라우저에서 직접 열어 실행합니다.

## 파일 구성

- **`index.html`** — "오늘의 명언" 페이지. 날짜 기반 명언 자동 선택, 랜덤 명언 버튼, 페이드 전환 효과 포함.
- **`tarot-blog.html`** — "달빛 타로" 블로그 레이아웃. 사이드바 + 메인 그리드 구조, 날짜 기반 오늘의 타로 카드 위젯, JS로 생성되는 별 배경 포함.

## 코드 패턴

**CSS 변수**는 `:root`에 정의되어 전체에서 사용됩니다. 전체 테마 변경 시 이 변수들만 수정하면 됩니다.

**날짜 기반 콘텐츠**: 두 페이지 모두 `new Date()`로 오늘 날짜를 구한 뒤 배열 길이로 나머지 연산하여 일별 콘텐츠를 결정합니다. 외부 데이터 소스는 사용하지 않습니다.

**외부 JS 의존성 없음** — 모든 인터랙션은 각 파일 하단 `<script>` 내 순수 바닐라 JS로 구현되어 있습니다.

**폰트**: `tarot-blog.html`은 `<style>` 내 `@import`로 Google Fonts(`Noto Serif KR`, `Cinzel`)를 불러옵니다. 올바른 렌더링을 위해 네트워크 연결이 필요합니다.
